How It Works

1. Configure Stripe

Paste your Payment Link URLs from the Stripe Dashboard. When a customer clicks a plan button, they are redirected straight to Stripe's hosted checkout — no API keys or server needed.

2. Test & Go Live

Stripe Payment Links can be created in test mode or live mode directly from your Stripe Dashboard. Use test links while developing, then swap in live links when you're ready.

3. Deploy

Hit "Deploy Now" to push all site files to a GitHub repo and enable GitHub Pages. The site will be live at https://<owner>.github.io/<repo>/ within minutes.
